# Lab 7 - Task 2: OOP in Python

## Description
This project demonstrates Object-Oriented Programming concepts in Python
using a shop product system.

## Project Structure
- `models.py` — contains Product, Food, and Toy classes
- `main.py` — demonstrates usage of the classes

## How to Run
```bash
python main.py
```

## Classes
- **Product** — base class with name, price, stock attributes
- **Food** — child class, adds calories and expiration date
- **Toy** — child class, adds material and recommended age

## OOP Concepts Used
- Inheritance
- Method overriding
- Polymorphism
- Modules and imports
